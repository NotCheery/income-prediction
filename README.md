# income-prediction

## 📌 Project Overview
This project predicts whether an individual's income is high or low using the Adult Census Dataset.

It is a supervised machine learning project focused on binary classification.

Key features used include: age, education-num, workclass, race, sex_selfID, hours-per-week

The target variable is: income_binary (0 = below 50K income, 1 = higher than 50K income)

## ⚙️ Data Science & Machine Learning Workflow
- Data Cleaning & Preprocessing

Cleaned and transformed data using Pandas and NumPy

- Feature Selection & Model Training

Selected key features relevant to income prediction

- Trained a Logistic Regression model using scikit-learn

- Model Evaluation

Evaluated performance with train-test split and classification metrics

## 📊 Tech Stack
Python (Pandas, NumPy, scikit-learn)

Jupyter Notebook for experimentation

Machine Learning: Logistic Regression (binary classification)
